# 🎵 Site Musical

Projeto de desenvolvimento web de um site para apresentação de artistas musical, criado com foco em design, organização visual e experiência do usuário.

## 📌 Sobre o projeto

Este projeto consiste em uma página web musical desenvolvida para apresentar informações sobre artistas, músicas, eventos e identidade visual.

O objetivo foi praticar conceitos de desenvolvimento front-end, como estruturação HTML, estilização com CSS e criação de uma interface moderna e responsiva.

## 🚀 Tecnologias utilizadas

- HTML5
- CSS3

## 🎨 Funcionalidades

✔️ Página inicial com identidade visual do artista 
✔️ Objetivos
✔️ Layout responsivo para diferentes telas
